juejin-download
===============

掘金小册下载

使用方法
----

### 安装依赖

npm install

### 获取 Cookie

-   安装 Chrome 或 Firefox 的 cookie editor 扩展
-   打开 掘金
-   打开扩展程序
-   点击右下角的 "Export" -> "Export as JSON" (保存到剪贴板)
-   把你剪贴板上的内容粘贴到 `cookies.json` 文件中

### 执行脚本

node main.js
