Sequelize
=========

> Sequelize 是一个现代的 TypeScript 和 Node.js ORM，适用于 Oracle、Postgres、MySQL、MariaDB、SQLite和SQL Server 等多种数据库。它提供了可靠的事务支持、关系映射、快速和延迟加载、读取复制等功能。

Development
-----------

# install dependencies
$ yarn install

# start dev server
$ yarn start

# build docs
$ yarn run build
