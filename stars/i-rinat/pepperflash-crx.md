PepperFlash updater
===================

Overview
--------

Downloads current version of PepperFlash plugin for Chrome OS from Chrome update servers.

Run
---

```
$ python3 pepperflash-update.py
```

The script will download `.crx3` file to the current directory. CRX is a Chrome extension format, but it can be unpacked with `unzip`.
