✔️ TypeScript 语言用于大规模应用的 JavaScript 开发。 ✔️ TypeScript 支持类型，是 JavaScript 的超集且可以编译成纯 JavaScript 代码。 ✔️ TypeScript 兼容所有浏览器，所有宿主环境，所有操作系统。 ✔️ TypeScript 是开源的。
