歌单助手
====

> 对多个平台的歌单进行备份、分析，发现都自己听过些啥

#### 支持平台

-   网易云音乐
-   虾米音乐

# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build
