Getting started
---------------

### Download

Via npm

$ npm install animejs --save

or manual download.

### Usage

#### ES6 modules

import anime from 'animejs/lib/anime.es.js';

#### CommonJS

const anime \= require('animejs');

#### File include

Link `anime.min.js` in your HTML :

<script src\="anime.min.js"\></script\>

### Hello world

anime({
  targets: 'div',
  translateX: 250,
  rotate: '1turn',
  backgroundColor: '#FFF',
  duration: 800
});
