### 本地开发

本站以 Docusaurus 3，一款现代化的静态网站生成器构建。

```
$ yarn  # 如果需要的话，安装或升级依赖
$ yarn build-helpers  # 如果变更了 scripts/asmdb 的话
$ yarn start
```

此命令会启动本地开发服务器，并打开一个浏览器窗口。 大部分变更都会自动触发窗口更新，不用手工重启服务器。
