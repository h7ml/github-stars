aura.nvim
=========

Aura colorscheme for Neovim written in Lua

Requirements
------------

-   Neovim >= 0.7.0
-   Treesitter (optional)

Supported Plugins
-----------------

-   TreeSitter
-   LSPDiagnostics
-   NvimTree
-   Telescope
-   Lualine
-   GitSigns
-   DiffView

Installation
------------

```
-- Using Packer
use 'helbing/aura.nvim'
```
