📑简介
====

猫抓(cat-catch) 资源嗅探扩展，能够帮你筛选列出当前页面的资源。

📘安装方法
======

应用商店安装
------

通过安装地址的链接到官方扩展商店即可安装。

源码安装
----

1.  Git Clone 代码。
2.  扩展管理页面 打开 "开发者模式"。
3.  点击 "加载已解压的扩展程序" 选中扩展文件夹即可。

crx安装
-----

1.  Releases **右键另存为**下载crx文件。
2.  扩展管理页面 打开 "开发者模式"。
3.  将crx文件拖入扩展程序页面即可。

📚兼容性说明
=======

1.0.17版本之后需要Chromium内核版本93以上。 低于93请使用1.0.16版本。 要体验完整功能，请使用104版本以上。
