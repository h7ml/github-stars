Bruno
=====

一套企业级移动端 Flutter 组件库

简体中文 | English

**Bruno 项目组诚邀用户参与调研反馈，我们珍视每一位用户的建议并纳入到后续规划中。同时为了增进 Bruno 用户间交流要求，我们建立了社区群，欢迎大家加入**👇

✨ 特性
----

-   提炼自企业级移动端产品的交互和视觉风格
-   开箱即用的高质量 Flutter 组件
-   提供满足业务差异的主题定制能力
-   设计工具赋能开发全链路

接入
--

Flutter 工程中 pubspec.yaml 文件里加入以下依赖：

dependencies:
  bruno: version
     

示例
--

import 'package:bruno/bruno.dart';
...
BrnBubbleText(
  maxLines: 3,
  text: 'I am first bubble text',
)
