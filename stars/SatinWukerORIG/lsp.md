LSP 老色批 命令行工具
=============

主理念
===

from 枪弹轨迹 (已三连)

前言
==

每当打开青年大学习时，总能在弹幕里看到，“学习新思想，争做新色批”，那时我就知道，当下的世人是多么需要接受新思想，是多么渴望躯体于心灵的解脱。许多人可能不理解我的努力，可能不明白我的用心良苦。但不论如何，我一定会告诉那个人，“让人心情愉悦是一件好事，也是一件正事”。这就是为什么我会坚定不移地站在这里，用一种不同寻常的方法向你展示着这五彩斑斓的世界！

介绍
==

-   每当运行时，lsp会从图库随机抽取一张图片
-   本铯钍库的大部分图片来自于 Pixiv, 部分来自于闲暇时收集的图片
-   一天只可抽取一张色图，不能抽多了，看多了伤身体:p 🤣
-   看完的图片会被立刻删除，因为我也不知道为啥哈哈哈
-   所有图片都已经过严格筛选，绝不涉黄违法，请大家放心使用！

运行
==

运行条件
----

-   运行 `pip install -r requirements.txt`
-   Linux 用户需安装 imagemagick: `sudo apt-get install imagemagick`

运行代码
----

lsp.py 为主程序，直接运行即可: `python lsp.py`
